# 深度學習與影像處理專題簡要內容
## 專題內容：
本專題的實作內容為：使用Qt designer實作出兩個圖形使用者介面而其中內容包含電腦視覺影像處理和深度學習兩大方向，在影像處理的部分為:利用opencv套件與自製演算法實現邊緣偵測、顏色萃取、各種特徵萃取與直方圖均衡化等各式影像處理，在深度學習的範疇內則是:利用Pytorch架構完成CIFAR10資料集多類辨識、MNIST手寫字辨識、貓狗二元辨識模型優化與資料前處理優化等內容。
